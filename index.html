数当てゲーム


ルール

1.ある自然数がお題の数になります。(現状256以上65536以下)
A natural number is given, and it is not shown. (now 256~65536)
2.お題の数より小さい自然数を入力すると、お題の数を入力された数で割った余りを表示します。
If you input a natural number smaller than the given number, the reminder from a division is shown.
3.お題の数ちょうどを入力すると、あなたの勝ちです。
If you input just the given number, you win.
4.お題の数より大きい自然数、またはお題の数の約数を入力すると、あなたの負けです。
If you input a natural number larger than the given number or being a divisor of it, you lose.
5.1回目の入力を除いて、入力できる数は直前に示された余りの倍数に限ります。
You can input only a multiple number of the latest divisor except the first input.
