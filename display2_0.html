数当てゲーム2


ルール

1.ある自然数がお題の数になります。(現状256以上65536以下)
A natural number is given, and it is not shown. (now 256~65536)
2.お題の数ちょうどを入力すると、あなたの勝ちです。
If you input just the given number, you win.
3a.お題の数より小さい自然数を入力するとお題の数と入力された数で互除法が実行されて、割り算の回数が表示されます。
If you input a natural number smaller than the given number, the mutual division method is proceeded with the input and the given and how many divisions are in the method is shown.
3b.互除法の過程で生成される余りの列が以前生成されたものを完全に含む場合、その部分も併せて表示されます。
If the sequence of divisors produced by the method in each input includes the whole of a sequence produced before, the part is also shown.
4.お題の数より大きい数を入力して余りの列が表示されると、あなたの負けです。それ以外はルール3と同様に処理されます。
If a sequence of divisors is shown when you input a natural number larger than the given number, you lose. If not, it is the same as the rule 3.
5.1度入力した自然数を再度入力することはできません。
You cannot input any natural number input before.
